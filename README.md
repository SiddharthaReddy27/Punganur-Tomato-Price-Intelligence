# Punganur Tomato Price Intelligence Dashboard

This project analyzes tomato price data from the Punganur market (Andhra Pradesh) and provides an interactive decision-support dashboard to help farmers choose the best time to sell their produce.

The goal of this project is to turn raw mandi price data into clear business insights using Python and Power BI.

## 📊 Project Overview

Tomato prices change drastically across the year due to:
- Seasonal supply
- Market demand
- Weather and harvest cycles

This project helps answer:

- Which months give the highest profit?
- Which months are risky due to high volatility?
- When should farmers sell quickly?
- When should they wait for better prices?

## 🔍 What This Project Shows

- Daily and monthly tomato price trends
- Seasonal patterns in the Punganur market
- Price volatility by month
- Risk vs profit comparison
- Farmer selling strategy (safe vs risky months)

---

## 📈 Key Insights

- Prices remain weak from **January to May**
- **July and August** form the first major profit window
- **November and December** have the highest prices but also the highest risk
- Farmers needing stability should prefer **January to March**
- Farmers who can take risk should target **July, August, November and December**
